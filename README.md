# Figma2Code Workshop

Workshop para converter designs Figma em codigo Angular usando GitHub Copilot.

## Setup

```bash
git clone <repo-url>
cd codefast
npm install
```

## Development

```bash
npx ng serve
```

Abre `http://localhost:4200/` no browser.

## Build

```bash
npx ng build
```
